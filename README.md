# Mini Backend User Management (Python)

Practical exercise for Boost - Back End Development.

## Description

This project implements basic backend logic for managing users and computing statistics using pure Python (standard library only).

The program:
- Loads user data from a JSON file
- Validates user fields (email, age, logins)
- Filters active and adult users
- Calculates statistics:
  - Average age
  - Total number of logins
  - User with the highest number of logins
- Handles file and JSON errors
- Prints a formatted console report

## Requirements

- Python 3.9+
- Standard library only

python3 main.py
